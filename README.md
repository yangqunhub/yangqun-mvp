# 杨泽群个人网站

健康复利导师 × 数字建造者 的个人品牌网站

## 快速开始

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建静态文件
npm run build
```

## 部署到 GitHub Pages

1. **推送代码到 GitHub**
   ```bash
   cd yangzequn-portfolio
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin https://github.com/yangqunhub/yangzequn-portfolio.git
   git push -u origin main
   ```

2. **启用 GitHub Pages**
   - 进入仓库 Settings → Pages
   - Source 选择 "Deploy from a branch"
   - Branch 选择 "gh-pages" / "root"

3. **自动部署**
   - 每次推送到 main 分支会自动部署
   - 访问 https://yangqunhub.github.io/yangzequn-portfolio/
